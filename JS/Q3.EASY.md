What is the output of the following code?


```js
const fruits = ['apple', 'banana', 'orange', 'grape'];

const x = fruits.slice(-2);

console.log(x); 
```

A. `[undefined, undefined]`  
B. `['banana', 'orange']`   
C. `['orange', 'grape']`    
D. `['apple', 'banana']`   

Answer: C. ['orange', 'grape']
